Устанавливаем депсы
npm i

Запускаем сервер
npm run start

Сервер висит на локалхосте на порте 3333

Ивенты сокета
'bci:fft' - работает, но не факт, что правильно
'bci:topo' - работает
'bci:time' - багает
'bci:signal' - работает
'bci:filter' - очень багает
'bci:motion' - заглушка под акселерометр
'SIGINT' - багает
